## Tech/ISV 파트너 지표 현황 및 추가 요청 목록

**Why:** STAR 시스템 개발 설계안(파트너전략그룹, 2026.03.30) 기반으로 사업건전성 대시보드에서 필요한 지표를 점검하고, 설계 단계에서 선제적으로 필드 추가 요청하기 위함.

**How to apply:** ❌ 항목은 파트너전략그룹에 필드 추가 요청 필요. 등급 유지 요건 달성률은 파트너 부서 문의 후 확정 예정.

| 등급 유지 달성률 및 요건 달성률 | **등급별 매출 목표금액**, 요건 관련 필드 없음 | ❌ 추가 요청 | 파트너 부서 문의 후 세부 확정 예정 |
| Global Reselling 매출 (지역별) | 매출 실적에 **국가/리전** 없음 | ❌ 추가 요청 | 대내/외는 그룹사 구분이지 국내/해외 아님 |
| 혜택 유형별 현황 | 내용 자유 텍스트, **유형 코드** 없음 | ❌ 추가 요청 | MDF/바우처/교육 등 코드화 필요 |
| 혜택 활용률 | **사용금액 or 활용여부** 없음 | ❌ 추가 요청 | |
| 리셀러 마진율 | **리셀러 마진(할인율, %)** 없음 | ❌ 추가 요청 | 사업부장 요청, 권한 관리 필요 |
| 파트너십 투자 대비 매출 ROI | **파트너십 체결비용(원)** 없음 | ❌ 추가 요청 | 사업부장 요청, 권한 관리 필요 |
