## 4가지 핵심 변경

| 변경 | 내용 |
|------|------|
| **변경 01** | Phase 0 신설 — 4개 과제 착수 전 AIPro/FabriX 기능 2주 내 확인 |
| **변경 02** | 이중 레이어 환경 — Windows PC(데이터) + 외부 서버(웹) |
| **변경 03** | Task 순서 역전 — Task 3 → 2 → 1 → 4 (리스크 낮은 순) |
| **변경 04** | 3인 역할 분담 — PM·데이터·프론트 + 공유 마스터 컨텍스트 |

## Task 추진 순서

| 순서 | 과제 | 이유 |
|------|------|------|
| 1st | **Task 3** 시안 생성 + 시뮬레이션 | DB 접근 없이 가상 데이터로 시작 가능, 리스크 최저 |
| 2nd | **Task 2** 데이터 모델 개선 | SQL 생성 능력 검증, Task 1의 백엔드 선행 |
| 3rd | **Task 1** 웹 대시보드 재구축 | 시안(Task 3) + SQL(Task 2) 결합 후 실구현 |
| 4th | **Task 4** 자동 분석 + 리포트 발송 | 환경 갖춰진 후 자동화 또는 반자동 검증 |
