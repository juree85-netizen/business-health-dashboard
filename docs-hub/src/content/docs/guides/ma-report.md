## 호출 방법

```
/ma-report [매수 기업] [피인수 기업]
```

**예시:**
```
/ma-report Coupa Rossum
/ma-report Thoma Bravo Medallia
/ma-report Salesforce Slack
/ma-report SAP LeanIX
```

**원문 자료 함께 제공 시:**
```
/ma-report Coupa Rossum

1. Coupa 개요
...
```
→ 원문을 1차 소스로 사용하고, 누락 데이터는 웹 검색으로 보완
