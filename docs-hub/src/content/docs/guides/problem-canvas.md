```

```

슬래시 명령 없이도 아래 표현으로 자동 트리거됩니다.

| 트리거 표현 | 예시 |
|-------------|------|
| `/problem-canvas` | `/problem-canvas 고객 이탈률이 높아지고 있다` |

| 어떻게 해결할까 | `매출이 정체된 이유와 해결 방법을 알려줘` |

## 실행 흐름

```
문제 입력
    ↓
[Phase 1] BMC 진단
  · 9개 블록 중 주요 문제 블록 특정
  · 연쇄 영향 블록 매핑
    ↓
[Phase 2] 맥킨지 7단계
  Step 1. 문제 정의 (SMART)
  Step 2. 문제 구조화 (Issue Tree)
  Step 3. 우선순위 설정
  Step 4. 분석 계획
  Step 5. 핵심 발견사항
  Step 6. 종합 (So What?)
  Step 7. 권고안 + Elevator Pitch
```

## BMC 9개 블록

| 블록 | 코드 | 핵심 질문 |
|------|------|-----------|
| 고객 세그먼트 | CS | 누구를 위한 사업인가? |
| 가치 제안 | VP | 어떤 문제를 해결해주는가? |
| 채널 | CH | 어떻게 고객에게 전달하는가? |
| 고객 관계 | CR | 고객과 어떤 관계를 맺는가? |
| 수익 흐름 | R$ | 무엇으로 돈을 버는가? |
| 핵심 자원 | KR | 무엇이 있어야 작동하는가? |
| 핵심 활동 | KA | 무엇을 해야 하는가? |
| 핵심 파트너십 | KP | 누구와 함께해야 하는가? |
| 비용 구조 | C$ | 무엇에 돈이 드는가? |

## 맥킨지 7단계 구조

| Step | 단계명 | 산출물 |
|------|--------|--------|
| 1 | 문제 정의 | SMART 문제 진술 + 성공 기준 |
| 2 | 문제 구조화 | MECE Issue Tree |
| 3 | 우선순위 설정 | 영향도 × 해결가능성 매트릭스 |
| 4 | 분석 계획 | 이슈별 필요 데이터·확인 방법 |
| 5 | 핵심 발견사항 | 논리 기반 Finding 3~5개 |
| 6 | 종합 | So What? — 핵심 메시지 서사 |
| 7 | 권고안 | 우선순위별 액션 + Elevator Pitch |
