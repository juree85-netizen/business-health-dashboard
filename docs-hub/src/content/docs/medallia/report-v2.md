## FCF 커버리지 추이 (3장)

| 연도 | FCF 추정 | 연이자 | FCF 커버리지 | 상태 |
|------|---------|--------|-------------|------|
| '21 | ~$90M | $135M | **0.67×** | 위험 ← 인수 당시 |
| '22 | ~$85M | $180M | **0.47×** | 구조적 부실 진입 |
| '23 | ~$120M | $300M | **0.40×** | 최악 구간 |
| '24 | ~$145M | $280M | **0.52×** | 비용 절감 일부 효과 |
| '25~'26 | ~$170M | $270M | **0.63×** | 경영권 이전 결정 |

※ LIBOR 기준금리 0.75% + credit spread 약 7.0% = 실효 이자율 약 7.5%. $1.8B × 7.5% ≈ $135M  
**핵심:** FCF 커버리지 1.0× 기준선을 단 한 번도 넘지 못한 채 경영권 이전.

## 이해관계자 관점 분석 (4장 신규)

### Thoma Bravo — PE 투자자 관점

| 단계 | 당시 판단 | 사후 평가 |
|------|----------|---------|
| 투자 논거 | EV/Revenue 9× 저렴. ARR 23~26% 고성장 | FCF 커버리지 0.67× 간과 — 저렴함이 아니라 위험 신호 |
| FCF 개선 기대 | TB 플레이북으로 3~5년 내 커버리지 1.5× 달성 | 금리 인상 속도 > FCF 개선 속도 — 플레이북 작동 불가 |
| 금리 리스크 | 헤지 불필요 판단 | SOFR 100% 노출 → $135M → $300M. 금리 스왑 미실행이 결정적 패착 |
| Exit 전략 | "3~5년 내 IPO 또는 전략 매각" | SaaS 멀티플 수축 + GenAI + 부채 $2.78B → 4가지 경로 전부 차단 |

### Blackstone — 채권자 관점

| 단계 | 당시 판단 | 사후 평가 |
|------|----------|---------|
| 대출 논거 | TB 보증 + Senior Secured 구조 | TB 추가 투자 거절로 보증 효과 소멸 |
| 변동금리 | 금리 상승 시 수익 자동 증가 | 이자 수익 증가 + 동시에 Medallia 상환 능력 파괴 — 자기 파괴적 역설 |
| PIK 허용 | 포트폴리오 생존 기간 연장 | 원금 $1.8B → $2.78B 팽창 — EV/Par 급락의 직접 원인 |
| D/E Swap | — | "원하지 않은 Equity" 보유. 실제 Exit 가치 < 원금 $2.78B |

### 이해관계자별 개선 기준

| 항목 | Thoma Bravo | Blackstone |
|------|------------|-----------|
| FCF 커버리지 기준 | ≥ 1.5× 이상만 LBO 진행 | ≥ 1.5× 이상 대출 승인 |
| 금리 헤지 | 변동금리 30~50% IRS 헤지 의무화 | 금리 캡 취득 요구 조항 삽입 |
| PIK 제한 | 비중 50% 이하, 기간 최대 2년 | 2년 이후 현금 이자 전환 의무화 |
| EV/Par 경보 | 2.0× 이하 → Equity 투입 또는 Exit 가속 | 1.5× 이하 → 조기 상환 또는 금리 재협상 |
| Exit 조건 | FCF 커버리지 2.0× 달성 시로 계약 명시 | 만기 내 Exit 미실현 시 조기 상환권 확보 |
| GenAI 평가 | LLM 대체 비중 ≥ 40% → 투자 제한 | GenAI 고위험 기업 spread +100bp |

## GenAI 역할 정의 (4장)

> GenAI는 Medallia의 직접적 현금 부족 원인이 아니다. 다만 **Exit 가능성을 구조적으로 차단한 핵심 요인**으로 작용했다 — 잠재 인수자의 지불 의향가를 낮추고, IPO 재진입 시 투자자 수요를 감소시켰다.
