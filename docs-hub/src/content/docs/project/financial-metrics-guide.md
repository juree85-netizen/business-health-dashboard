## 주요 내용

- **Waterfall 시각 다이어그램**: 손익계산서(발생주의) → 현금흐름표(현금주의) 흐름을 색상과 화살표로 표현
- **D&A 왕복 여행**: EBITDA→EBIT에서 차감, Net Income→OCF에서 재가산하는 이유 상세 해설
- **운전자본 변동**: 매출채권(−) · 재고자산(−) · 매입채무(+) 각각의 이유를 표와 원칙으로 정리
- **FCF vs CFF 구분**: 가장 흔한 혼동 포인트를 원천(FCF)·배분(CFF) 관계로 명확히 정리
- **실전 Q&A 7문항**: 영업이익=EBIT 여부, CAPEX vs D&A 차이, 재고자산을 왜 빼는가 등 실무 혼동 포인트 해설
