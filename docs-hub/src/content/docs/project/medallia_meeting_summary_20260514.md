## 핵심 요약 (3줄)

- Medallia 실패의 본질은 **고금리·PIK 누적·성장 정체의 3중 압박**이며, GenAI에 의한 사업 대체 가능성이 구조적 취약점으로 확인됨
- Thoma Bravo → Blackstone 경영권 이전은 PE 입장에서 "담보 실행"이며, 양측 실손 규모는 FCF 데이터 확보 후 정밀 재계산 필요
- SaaS 밸류에이션 정상화 국면에서 M&A 저점 매수 타이밍이 도래하고 있으며, 평가 기준(NRR 지속성·기술 대체 위험·FCF 커버리지)을 지금 정립해야 함

## 2. 보고서 품질 검토 — 리더 피드백 요지

### 분석 기준 통일 지시

- FCF를 일관된 분석 기준으로 사용 — EBITDA·영업이익으로 중간에 기준을 바꾸지 말 것
- "FCF가 나와야 이자 커버리지, Par Coverage, 투자 수익성이 한꺼번에 정리된다"고 강조
- EBITDA ≠ FCF 혼용 시 분석 논리가 무너진다고 반복 지적

### 데이터 정합성 요구

- 인수 당시 매출($0.7B) → 현재 매출 성장률을 역산(Hindcast)하여 실제 성장 궤적 확인
- 상장 폐지 이후 재무 데이터 미공개 → Substack·Medium 등 영문 서드파티 분석글 검색 지시
- "로직이 다 맞아떨어져야 한다" — 개별 수치가 아니라 전체 이야기(Story)가 일관되어야 함

### 보고서 목적 재정의

- 단순 사실 정리를 넘어 **시사점 도출 → M&A 평가 기준 정립**까지 연결
- "개선점을 고민하는 게 아니라, 우리가 인수할 때 무슨 기준으로 봐야 하는지 배우는 것"
- 콘설팅 관점이 아닌 사업주(사장) 관점으로 문제를 바라볼 것을 주문

## 3. 재무 수치 확인 및 해석 논의

| 항목 | 회의 중 언급 수치 | 해석 및 맥락 |
|------|-----------------|-------------|
| 인수 당시 매출 | ~$0.7B | 상장 폐지 이후 최근 수치 미공개. 역산 필요 |
| 인수가 (EV) | ~$6.4B | Equity $4.6B + Debt $1.8B 구조 (Thoma Bravo LBO) |
| EV/Revenue 배수 | ~9× | 당시 SaaS 시장 평균 대비 "싸게 산" 수준 |
| Rule of 40 (현재) | 10.8 | SaaS 건전성 기준선(40) 대비 현저히 낮음 |
| 연간 이자 부담 | ~$0.3B/년 | PIK 포함 변동금리 구조. 금리 상승으로 2.8B까지 누적 |
| FCF | 미공개 (적자 추정) | **FCF < 연간 이자 — 이것이 파국의 수학적 근거** |
| Blackstone 채권 원금 | $1.8B → 누적 ~$2.8B | 변동금리 인상으로 채무 누적. 경영권으로 전환 |
| Thoma Bravo 자기자본 | ~$4.8B (추정) | 실질 손실액은 이자 납부분 차감 후 계산 필요 |

**핵심 수식:**
```
연간 이자 지출 ≈ $0.3B/년
FCF < 이자 지출 → FCF 커버리지 < 1.0× → 구조적 부실
3.1년 × $0.3B ≈ $0.93B 누적 이자 납부 후 경영권 상실
```

## 4. 실패 원인 심층 분석

### ① 고금리 — 가장 직접적 원인

- SOFR 연동 변동금리 → 금리 인상기에 이자 부담이 $1.8B → $2.8B으로 급증
- PIK 비중을 중간에 축소 시도했으나 이미 누적된 부채 구조를 되돌리기 어려웠음

### ② 성장 정체 — 예측 실패

- 인수 당시 매출 성장률 예상: 23~26%p → 실제 성장률 훨씬 작았던 것으로 확인
- Rule of 40 = 10.8 → 성장(G)과 수익성(P) 모두 악화된 결과

### ③ 사업 대체 가능성 (GenAI 위협) — 구조적 취약점

- Medallia 사업 = VoC 수집 + 서베이 + 분석 → "이 영역은 대체 가능하다"
- GenAI 등장으로 기존 Survey/VoC SaaS의 차별점이 빠르게 소멸
- SAP 연계 협력 시도도 실패 → 외부 생태계 의존 전략 한계

> **실패의 3중 압박 구조:** 고금리(외부) + PIK 누적(구조) + 성장 정체(사업)  
> 세 가지가 동시에 작용해 개별 대응이 불가능했던 케이스. GenAI 가속화로 반전 가능성까지 차단 → 엑시트 실기(失期)

## 5. Blackstone·Thoma Bravo 포지션 분석

### 경영권 이전의 구조적 해석

- Blackstone은 채권자(Lender) 포지션 → Thoma Bravo가 채무 불이행 시 담보(경영권) 실행
- "얘네들이 Blackstone에 돈을 빌린 거잖아. 얘는 그 원금을 회사로 가져간 거지, 돈으로 못 받은 거지"

| 주체 | 투입 자원 | 회수 내용 | 실질 손익 해석 |
|------|-----------|-----------|----------------|
| Thoma Bravo | Equity ~$4.8B | 경영권 상실 | 이자 납부분 제외 실손. Equity 전액 손실에 가까움 |
| Blackstone | Debt $1.8B | 경영권(지분) 가치 ~$2.8B? | 경영권 가치 = $2.8B이면 수익, 실질 가치 미달 시 손실 |

**리더의 비유적 설명:**
> "내가 상가를 샀어. 장사가 잘 안 돼서 한 달에 100만원씩 3년 동안 썼는데, 그냥 담보로 은행에 준 거야. 자기네들이 엄청 불쌍한 거 없어요."  
> → **PE의 LBO는 지자본 손실이 '빌린 돈'으로 이루어졌다는 점을 항상 고려해야 함**

## 6. M&A 전략 시사점 및 논의

### 인수 타이밍 판단

- "SaaS 밸류에이션 정상화 → 지금이 저점 인수 타이밍이 온다" — 리더 직접 발언
- GenAI 영향으로 핫스팟이 이동 중 → 사업 본질 판단이 더 중요해지는 환경

### M&A 평가 기준 (이번 케이스에서 도출)

| 평가 항목 | Medallia에서의 문제 | 향후 기준 |
|-----------|-------------------|-----------|
| FCF 커버리지 | FCF < 이자 → 구조적 부실 | FCF ÷ 이자비용 ≥ 1.5× 이상 확인 |
| NRR 지속성 | 성장 정체·NRR 하락 | NRR 110%+ 지속 가능 여부 + 업셀링 경로 확인 |
| 기술 대체 위험 | VoC/Survey → GenAI 대체 | GenAI 대체 불가 영역인지 + 기술 방어력 평가 |
| 금리 시나리오 | 변동금리 + 고금리 미대응 | 고금리 지속 시 3년 FCF 시뮬레이션 필수 |
| 시장 타이밍 | 밸류 피크에 인수 | EV/Revenue 정상화 국면 진입 여부 확인 |

### 자사 M&A 제약 조건

- PS(주가·실적) 영향 우려로 공격적 M&A 결정이 어려운 내부 구조
- "중간 지분 투자가 안 된다. 전부 다 해버려야 한다. 그러면 돈 나온 데 사야 한다"
- "M&A가 쉬운 게 아니고 우린 제약이 너무 많다" — 리더 발언

## 9. 종합 평가 및 시사점

**보고서 분석의 핵심 교훈:**

- PE LBO의 실패는 이자 부담 단독이 아닌, **성장 정체 + 금리 상승 + 사업 대체 가능성의 복합 작용**으로 발생
- 밸류에이션 배수(EV/Revenue 9×)가 "싸 보여도" FCF 미래 추정이 틀리면 의미 없음
- PIK 이자는 현금 부담을 단기 유예할 뿐 — 근본 사업 가치 창출이 없으면 시한폭탄

**자사 적용 관점:**

- M&A 대상 평가 시 FCF 커버리지·NRR·GenAI 대체 가능성을 3대 필수 지표로 활용
- 고금리 환경 지속 가능성을 가정한 스트레스 테스트 시나리오 병행 필요
- 사업 본질(대체 불가 영역 여부) 판단 없이 재무 지표만으로 인수 결정 시 동일 위험 반복 가능

**리더 발언 인용 — 이번 스터디의 진짜 목적:**
> "M&A를 지금부터 진짜 열심히 관찰해야 되고, 그걸 하기 위해서 어떤 준비를 해야 하고,  
> 우리가 내년에 한다고 대상을 고를 때 무슨 기준으로 봐야 하는지도  
> 이 보고서에서 배워서 정리하자. 이런 거야."  
> → **Medallia 케이스 = 실패 분석이 아니라 M&A 평가 역량을 키우기 위한 훈련**
