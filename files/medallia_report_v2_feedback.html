핵심 관점 (v2.0 수정)
본 보고서는 Medallia 실패를 "금리 인상이라는 외부 충격에 의한 피해"가 아닌, "인수 당시부터 FCF 커버리지가 1.0× 미만이었던 구조적 설계 실패"로 재해석한다. 금리 인상은 방아쇠(Trigger)였고, 근본 원인은 취약한 현금창출력을 고부채 LBO 구조 위에 올린 판단 착오였다. 이 관점에서 GenAI 대체 위협, Exit 실패, 추가 투자 거절은 모두 하나의 인과 사슬로 연결된다.
1. 보고 개요

분석 배경

  • Thoma Bravo는 Medallia를 $6.4B(EV/Revenue 약 9배)에 인수하며 상장폐지 ('21.7월)
  • 美 연준 금리 인상으로 SOFR가 약 5.4%까지 상승, 차입 이자 부담 급증 ('22~'25년)
  • 연간 이자비용($0.3B)이 EBITDA($0.2B)를 초과하는 구조적 부실 직면, 채권단으로 경영권 이전 ('26.4월)
  • 한편, 역산 분석 결과 인수 당시 이미 FCF 커버리지가 1.0× 미만이었음이 확인됨 — 금리 상승은 trigger이지 root cause가 아님

분석 범위

  • 분석 대상: Medallia / Thoma Bravo
  • 분석 기간: '21년 인수 ~ '26년 채권단 경영권 이전
  • 재무 수치 출처: Bloomberg, OnlyCFO 분석 기반 추정치 (상장폐지 후 재무 비공개로 역산 추정값 포함)

분석 목적

  • PE 기반 SaaS LBO 실패 메커니즘 심층 분석 — 외부 충격론의 한계 재검토
  • FCF 커버리지·EV/Par Coverage 중심의 SaaS 기업가치 평가 기준 정립
  • GenAI 확산 이후 SaaS 사업 지속성 평가 방법론 제시
  • 자사 M&A 검토 시 활용 가능한 평가 프레임워크 도출 (별첨 6)

3. Medallia 투자/인수 현황

Thoma Bravo의 Medallia 인수 ('21년) — 인수 판단 근거와 내재 위험

  • EV/Revenue 멀티플 약 9배 적용 — 당시 SaaS 고성장 기업 평균(15~20배) 대비 "저렴한 인수"로 판단
  • ARR 성장률 23~26%, Gartner 리더 포지션, Fortune 500 고객군 기반 신뢰도 고려
  • 단, Rule of 40·FCF 마진이 업계 하위 10% 수준임을 인지했음에도 진행 → 당시 저금리 환경에서는 합리적으로 보였으나, 사후적으로는 FCF 전환 가정이 지나치게 낙관적이었음이 확인된다

핵심: 인수 첫 해('21년)부터 FCF 커버리지가 0.67×였다. 금리 인상 이전에 이미 구조적으로 취약한 상태였으며, 금리 상승은 이 취약성을 파국으로 전환한 트리거였다.

4. 주요 분석
관점 전환
기존 관점 (v1.0) 고성장 SaaS를 고멀티플로 인수했는데 금리가 올라서 망했다.
→ 외부 충격(금리 인상)이 주원인
수정 관점 (v2.0) 인수 당시부터 FCF 커버리지 0.67×인 기업을 변동금리 LBO로 샀다.
금리 상승은 트리거였고, 근본 원인은 현금창출력 과대 추정이었다.
→ 구조적 설계 실패 + 외부 충격의 복합 작용

실패 원인 — 심층 재분석

①
성장률 과대 추정: 고성장 지속 전제로 9배 멀티플(EV/Revenue) 적용
  • 인수 당시 ARR 성장률 23~26% 가정 — 실제 성장률은 훨씬 낮았던 것으로 추정 (상장폐지 후 비공개)
  • Rule of 40 = 10 내외임을 인지했으나 "성장세 지속 시 개선 가능"으로 낙관 — 실제로는 개선되지 않음
  • 단순 성장률 과대 추정이 아니라, FCF 전환 속도를 과대 추정한 것이 본질적 오류
②
인수 당시부터 FCF 커버리지 1.0× 미만
  • 역산 추정 결과 인수 첫 해('21년) FCF ≈ $90M, 이자 $135M → 커버리지 0.67×
  • 이는 금리 인상 없이 기준금리 동결만 지속됐어도 FCF 성장 없이는 5~7년 내 유사한 문제에 봉착할 구조
  • 결론: 금리 인상이 없었다면 망하지 않았을까? 아니다. 금리 인상이 없었다면 더 천천히 망했을 것이다
③
취약한 부채 구조: 변동금리 + PIK 복리 누적
  • SOFR 연동 변동금리 → 금리 인상기 이자비용 $135M → $300M 급등
  • PIK 비중 100%('21~'22) → 현금 부담은 줄었으나 원금이 $1.94B → $2.78B으로 복리 팽창
  • 추가 M&A 관련 차입 $0.6B 반영으로 총 Loan Par $2.78B 도달 — EV 대비 역전 상황 진입
④
이자부담으로 인한 상품 경쟁력 저하 → 선순환 붕괴
  • FCF 전액을 이자 상환에 투입 → R&D·AI 기능 고도화 투자 여력 소멸
  • Qualtrics(SAP 계열)·Salesforce의 AI 기능 강화 대비 Medallia 기능 정체 → 고객 이탈 가속
  • 고객 이탈 → 매출 성장 둔화 → FCF 개선 불가 → 이자 감당 불가 → 악순환 고착

Medallia 사업 구조와 GenAI 대체 가능성 분석

전제: GenAI는 Medallia의 직접적 현금 부족 원인이 아니다. 다만 Exit 가능성을 구조적으로 차단한 핵심 요인으로 작용했다 — 잠재 인수자의 지불 의향가를 낮추고, IPO 재진입 시 투자자 수요를 감소시켰다.

단순히 "AI 경쟁 심화"가 아니라, Medallia의 가치 레이어별로 대체 가능성이 다르다는 점을 구분해야 한다.

Exit 실패의 구조적 원인

v1.0에서는 "협상 난항"으로만 기술됐으나, 왜 협상이 불가능했는지 구조를 분석한다.

①
LBO 매물의 딜레마 — 잠재 인수자 입장
  • 전략적 인수자가 Medallia를 인수하려면: 잔존 EV(~$2.2B) + 부채 인수($2.78B) = 실질 비용 ~$5B
  • 그런데 GenAI 위협 받는 VoC SaaS에 $5B을 지불할 전략적 인수자는 없음
  • PE 재인수도 불가 — 이미 고부채 구조가 알려진 상황에서 레버리지 추가 불가
②
IPO 재진입 요건 미충족
  • 상장 가능 조건: 안정적 FCF + 성장률 회복 + 부채 구조 정상화 → 세 가지 모두 미충족
  • SaaS 멀티플 수축기('22~'25년)와 겹쳐 적정 공모가 설정 불가 — 투자자 수요 확보 어려움
③
자본구조 재조정 실패
  • 채권단과 헤어컷(부채 감면) 협상 시도 → Blackstone 등 채권자도 원금 손실 거부
  • 결국 Debt-to-Equity Swap이 채권자에게 최선인 상황으로 수렴
구조 요약: [부채 $2.78B 안고 팔 수 없음] + [IPO 요건 미충족] + [추가 자본조달 불가] + [채권자 헤어컷 거부] → 4가지 탈출구가 모두 막힘 → 경영권 이전이 유일한 결말

Thoma Bravo의 추가 투자 거절이 의미하는 것

"Thoma Bravo Declines to Invest More in Troubled Medallia" (Bloomberg, '26.4월) — 이 결정의 함의

  • PE 펀드 구조상 이유: 특정 포트폴리오에 추가 투자는 LP(출자자)에게 별도 설명 필요. 이미 손실 확정 포지션에 "good money after bad" 투입은 LP 신뢰 손상
  • 재무적 이유: 추가 투자해도 EV/Par Coverage 회복 불가능한 수준이었음 — $500M 추가 투입해도 커버리지 1.0× 미달
  • 신호의 의미: 이 거절 결정이 내려진 시점부터 경영권 이전은 기정사실. 즉 Bloomberg 기사가 나온 시점이 실질적 종료 선언이었음

5. 시사점

① SaaS 기업가치 평가 — 관점 전환

기존 평가 방식의 함정v2.0 교정 관점
EV/Revenue 멀티플이 낮으면 저평가 FCF 커버리지 < 1.0×이면 낮은 멀티플도 의미 없음 — Medallia 9×는 싼 게 아니었음
EBITDA 흑자 = 수익성 있음 EBITDA ≠ FCF. D&A·SBC가 큰 SaaS는 EBITDA 흑자여도 FCF 적자 가능 (별첨 5 참조)
ARR 성장률이 높으면 Future Value 있음 NRR이 FCF로 전환되는 속도가 핵심. 성장이 있어도 FCF 전환이 느리면 고부채 구조와 충돌
AI 기능 있으면 경쟁력 있음 핵심 가치가 "데이터 인프라"인가 "분석 레이어"인가 구분 — 후자는 GenAI 대체 고위험
Exit 계획 = "시장 좋아지면 IPO 또는 매각" "시장 상황에 의존"하는 Exit = Exit 계획 없는 것. 부채 구조 내 Exit 가능 조건을 사전 명시해야

② 향후 M&A 검토 시 필수 체크리스트

단계체크 항목Medallia 사례 결과최소 기준(안)
현금창출력 FCF 커버리지 (FCF ÷ 예상 이자) 인수 당시 0.67× — 미달 ≥ 1.5×

③ 사업건전성 대시보드 운영 방향

  • 잉여현금흐름(FCF) 지표 시범 산출 → 사업부 관리 협의 → 대시보드 적용 기준 확정
  • Rule of 40·NRR·FCF 통합 모니터링 체계 구축 — 단일 지표가 아닌 복합 지표 기준 경보 체계
  • EV/Par Coverage 내부 기준: 2.0× 이하 주의 / 1.5× 이하 경보 / 1.0× 이하 위험 (Medallia 사례 기준 역산)

※ FCF·EV 관련 수치는 Bloomberg, OnlyCFO 기사 기반 역산 추정치. 상장폐지 이후 재무 정보 비공개로 일부 추정 포함.

※ M&A 최소 기준(안)은 자사 내부 적용을 위한 초안이며, 시범 산출 후 사업부 협의를 통해 확정 필요.

별첨 5. EBITDA vs FCF — 왜 다른가

보고서 내에서 "EBITDA 흑자인데 현금흐름 적자"가 반복적으로 언급된다. 이 개념을 정확히 이해하기 위한 해설.

산출 구조 비교
지표산출 방법Medallia 추정 적용
EBITDA 순이익 + 이자비용 + 법인세 + 감가상각(D&A) ~$200M ('25년 추정) — 표면상 흑자
CFO
(영업현금흐름)		 순이익 + D&A + SBC(주식보상) ± 운전자본 변동 ~$250~300M 추정 (SBC 포함)
FCF CFO – CAPEX(설비투자) ~$170M 추정 — 이자 $270M보다 작음
FCF after Interest FCF – 현금 이자 지급액 ~–$100M — 실질 현금 적자
왜 SaaS는 EBITDA와 FCF 괴리가 큰가
  • SBC(주식기반보상): 임직원 스톡옵션·RSU는 비현금 비용으로 EBITDA에서 제외하지만, 기업 희석 효과는 실재함
  • 운전자본: 연간 선불 계약(Annual Upfront) 비중 높은 SaaS는 이연수익이 누적되어 CFO와 EBITDA 괴리 발생
  • CAPEX: 클라우드 인프라 확장 투자는 FCF를 직접 줄임
결론: Medallia처럼 순손실이 크고 SBC가 높은 SaaS는 EBITDA가 양수여도 실제 FCF는 음수일 수 있다. LBO 분석에서 EBITDA를 현금창출력 지표로 사용하면 이자 커버리지를 과대 추정하게 된다. 반드시 FCF 기준으로 이자 커버리지를 계산해야 한다.
