1. 속도 개선 후 화면 이슈

  • 기존 설계와 달라진 부분 다수 발견 (오른쪽 정렬, 기준 연월 표시, 한 화면 레이아웃 등)
  • 액션: 솔루션전략이 이슈 리스트업 후 메일 공유 예정

2. 기준 연월 오류

  • 화면에 4월이 아닌 3월로 표시됨 (4월 실적 미반영)
  • 액션: 회의 후 즉시 확인

4. 6월 개발 지연 예고

  • 원인 2가지:
    • 데이터 레이크 전체 개인정보 암호화 작업 (6월 말 마감)
    • NiFi → Airflow 전환 작업 (1월부터 지연되어 6월 마감)
  • 결론: 단순 작업(엑셀 업로드 등)은 우선순위 조정하여 진행, 복잡한 작업은 일정 공유 필요
