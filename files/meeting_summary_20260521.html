# 담당 내용 기한
1 전체 6월 개발 지연 감안하여 사업기회·VoC 6월 2주 내 완료 목표 관리 6월 2주차
2 솔루션전략 속도 개선 후 달라진 화면 이슈 리스트 메일 공유 회의 후 즉시
3 솔루션전략 4월 실적 기준 연월 오류 확인 회의 후 즉시
4 SDSC 사업기회 뷰 테이블 완료 일정 확인 후 메신저 공유 회의 후 즉시
5 솔루션전략 재무_시장·고객·상품 엑셀 파일 재공유 회의 후 즉시
6 SDSC 중복 데이터 샘플 담당자에게 공유 추후
7 SDSC VoC 오은성 프로님 인터페이스 작업 일정 업데이트 공유 추후

이슈 사항

  • 원인 2가지:
    • 데이터 레이크 전체 개인정보 암호화 작업 (6월 말 마감)
    • NiFi → Airflow 전환 작업 (1월부터 지연되어 6월 마감)
  • 결론: 단순 작업(엑셀 업로드 등)은 우선순위 조정하여 진행, 복잡한 작업은 일정 공유 필요
