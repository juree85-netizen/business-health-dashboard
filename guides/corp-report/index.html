섹션포함 조건
1. 산업/시장 현황항상 포함

2. 사업 실적 표5~10년 재무 추이 (데이터 있을 때)
3. 현지 법인 현황대상 국가 지정 시 추가
4. 시장 내 협력/전략 현황현지 활동 있을 때
5. 조직 관계·의사결정 구도데이터 있을 때
6. 삼성-[기업] 협업 이력협업 이력 있을 때
별첨 1. 해외 법인 현황분량 클 때
별첨 2. 주요 임원 프로필임원 데이터 있을 때
