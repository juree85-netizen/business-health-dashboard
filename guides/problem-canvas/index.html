/problem-canvas — BMC × 맥킨지 7단계 문제해결 스킬

사업의 문제점을 입력하면 BMC로 문제 위치를 특정하고, 맥킨지 7단계로 해결 경로를 도출합니다.
Phase 1(진단) → Phase 2(해결) 두 단계로 구조화된 분석을 제공합니다.
