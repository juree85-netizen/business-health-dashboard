수작업 데이터 업로드

시장 데이터 변환 파일

섹션 제목: “시장 데이터 변환 파일”

수기(수작업)로 작성된 시장 데이터를 DB 업로드용 포맷(Long format)으로 변환한 파일입니다.

파일 정보

섹션 제목: “파일 정보”
항목내용
대상 상품Brity Automation, Knox (All), Nexprime HCM, Nexprime SCM (All), EMM, ZTM
기준 연도2023 ~ 2028 (upload_year: 2025)
출력 행수2,178행 (198 소스 행 × 11 카테고리)
파일 형식TSV (탭 구분, UTF-8 BOM)

다운로드

섹션 제목: “다운로드”

⬇ market_data_converted.tsv

컬럼 구조 (기존 틀 포맷)

섹션 제목: “컬럼 구조 (기존 틀 포맷)”
컬럼설명
year데이터 기준 연도
category지표 구분 (tam, ext_sam, rnd 등)
value값 또는 수식/출처 문자열
product상품명
domestic_globalDomestic / Global
on_prem_cloudOn-prem / Cloud
biz_typeSI/SM / Solution / MSP / SaaS
ww010상품 코드
upload_year입력 기준 연도

카테고리별 value 처리 방식

섹션 제목: “카테고리별 value 처리 방식”
categoryvalue 처리
tam수기 입력값 (억원)
ext_sam수기 입력값 (억원)
rnd수기 입력값 (억원)
tam_grw자동 계산: (TAM_N - TAM_N-1) / TAM_N-1
sam_grw자동 계산: (SAM_N - SAM_N-1) / SAM_N-1
revenueDB 출처: l_sap.orderlist > vf219
sales_marketingDB 출처: l_sap.orderlist > vf599
revenue_grw수식: (vf219_N - vf219_N-1) / vf219_N-1
rnd_rto수식: rnd / vf219 * 100
sales_rto수식: vf599 / vf219 * 100
marketshare수식: vf219 / TAM

Excel에서 열기

섹션 제목: “Excel에서 열기”
  1. 위 링크로 파일 다운로드
  2. 확장자 .tsv → .txt 로 변경
  3. Excel에서 열기 → 텍스트 마법사 실행
    • 원본 형식: 구분 기호로 분리됨
    • 파일 원본: 65001: Unicode (UTF-8)
    • 구분 기호: 탭
  4. 마침 → 확인
