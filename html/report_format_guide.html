1. 페이지 설정 (용지·여백·방향)
□ 설정 경로: 레이아웃 탭 > 페이지 설정 > 여백·용지
항목설정값비고
용지 크기A4 (210 × 297 mm)
방향세로 (Portrait)
여백 — 위 / 아래25 mm
여백 — 왼쪽 / 오른쪽20 mm
머리글 여백15 mm
바닥글 여백8 mm페이지 번호 위치 기준
2. 기본 폰트 & 타이포그래피
□ 설정 경로: 홈 탭 > 글꼴
□ 기본 글꼴: 바탕체 (Batang)
환경글꼴 이름비고
Windows 한글 버전바탕 / 바탕체OS 기본 제공 — 별도 설치 불필요
Windows 영문 버전Batang / BatangChe동일 글꼴, 영문 이름으로 표시
한컴오피스 설치 시함초롬바탕바탕체 대체 가능 (자형 유사)
HY 폰트 보유 시HY바탕바탕체 대체 가능
       ※ Word 글꼴 검색 시 "바탕"으로 검색하면 바탕·바탕체·BatangChe 모두 표시됩니다.
□ 요소별 서식 정의
요소글꼴크기스타일정렬
제목바탕체20 pt굵게 + 밑줄가운데
보고 날짜바탕체14 pt보통오른쪽
요약 문단바탕체14 pt보통왼쪽
(1행3열 표 중간 셀)
소제목 Lv1바탕체14 pt굵게왼쪽
□ 소소제목 Lv2바탕체14 pt보통왼쪽
- 내용 Lv3바탕체14 pt보통왼쪽
· 내용 Lv4바탕체14 pt보통왼쪽
※ 보충 Lv5바탕체10 pt보통왼쪽
□ 요약 문단 표 설정 방법
- 삽입 > 표 > 1행 3열 삽입
- 중간 셀 선택 → 표 디자인 > 테두리 > 테두리 없음 적용
- 중간 셀에 요약 내용 작성 (좌·우 셀은 여백 확보용, 내용 없음)

3. 단락 설정 (간격·줄간격)
□ 설정 경로: 홈 탭 > 단락 > 줄 및 단락 간격 또는 단락 대화상자 (Alt + H + PG)
□ 기본 간격 (제목·날짜·요약·Lv1)
요소줄 간격단락 앞단락 뒤비고
제목1줄 (Single)0 pt12 pt
보고 날짜1줄 (Single)0 pt9 pt
요약 문단1줄 (Single)3 pt6 pt
소제목 Lv11줄 (Single)0 pt12 pt요약 문단 바로 다음: 앞 16 pt
□ 본문 레벨별 단락 뒤 간격 — 다음 줄 레벨에 따른 조건
현재 레벨기본 뒤 간격다음 줄 조건에 따른 뒤 간격
Lv2 (□) 16 pt 동일 Lv2 줄바꿈 → 9 pt  /  다음 Lv1 → 20 pt
Lv3 (-) 16 pt 동일 Lv3 줄바꿈 → 9 pt  /  다음 Lv4 → 12 pt  /  다음 Lv1~Lv2 → 20 pt
Lv4 (·) 12 pt 동일 Lv4 줄바꿈 → 9 pt  /  다음 Lv3 → 16 pt  /  다음 Lv1~Lv2 → 20 pt
Lv5 (※) 12 pt 동일 Lv5 줄바꿈 → 9 pt  /  다음 Lv4 → 12 pt  /  다음 Lv3 → 16 pt  /  다음 Lv1~Lv2 → 20 pt
       ※ 줄 간격을 고정값(Exactly)으로 설정하면 글자가 잘릴 수 있습니다. 모든 레벨은 1줄(Single) 옵션을 사용하세요.

4. 글머리 기호 계층 구조
□ Lv1~Lv5 기호 및 입력 방법
레벨기호입력 방법비고
Lv 11.숫자 및 마침표 직접 입력자동 번호 매기기 사용 안 함
Lv 2□직접 입력 (유니코드 25A1 + Alt+X)공백 1칸
Lv 3-하이픈 직접 입력공백 3칸 / 자동 대시 변환 주의
Lv 4·직접 입력 (유니코드 00B7 + Alt+X)공백 5칸
Lv 5※직접 입력공백 7칸 / 10pt / 보충·주석
□ 나열 기호 사용 규칙
- Lv3~Lv4에서 동일 수준 내용 나열 또는 순서 표시 시 1) 2) 3) 또는 ① ② ③ 사용 가능
- 이 경우 기존 글머리 기호(- · 등)는 한 단계씩 하위 레벨로 내려감
       ※ 하이픈(-)이 대시(—)로 자동 변환되는 경우: 파일 > 옵션 > 언어 교정 > 자동 고침 옵션 > "하이픈을 대시로" 해제
5. 들여쓰기 & 줄바꿈 규칙
□ 5.1 들여쓰기 설정값
레벨기호왼쪽 들여쓰기내어쓰기 (Hanging)비고
Lv 11.공백 0칸없음직접 입력, 자동 번호 매기기 사용 안 함
Lv 2□공백 1칸없음일반 공백(Space) 사용
Lv 3-공백 3칸없음일반 공백(Space) 사용
Lv 4·공백 5칸없음일반 공백(Space) 사용
Lv 5※공백 7칸없음일반 공백(Space) 사용
       ※ 내어쓰기(Hanging Indent)를 적용하지 않습니다. 모든 들여쓰기는 일반 공백(Space)으로 처리합니다.
□ 5.2 줄바꿈 정렬 원칙
- 한 줄 안에 내용이 마무리되지 않을 경우 Enter로 강제 개행
- 동일 레벨 줄바꿈 후 왼쪽 들여쓰기는 해당 레벨의 공백 수에 2칸 추가
· 예시: Lv3(-) 줄바꿈 후 → 공백 5칸 (3칸 + 2칸 추가) 으로 맞춤 정렬
· 예시: Lv2(□) 줄바꿈 후 → 공백 3칸 (1칸 + 2칸 추가) 으로 맞춤 정렬

8. 작성 전 체크리스트
□ 페이지 설정: A4 / 좌우 여백 20 mm / 바닥글 여백 8 mm 확인
□ 제목: 바탕체 20pt · Bold · 밑줄 · 가운데 정렬 적용
□ 날짜 형식
- 보고 날짜: 'YY.MM.DD 형식 / 오른쪽 정렬
- 본문 중 날짜: 'YY.M월 또는 'YY/MM/DD 형식
□ 요약 문단: 1행 3열 표 삽입 · 중간 셀 테두리 없음 · 14pt 왼쪽 정렬
□ 소제목: 14pt Bold · 자동 번호 매기기 해제 · 숫자+마침표 직접 입력
□ 계층 순서: 1. → □ → - → · → ※ 준수 (레벨 건너뛰기 금지)
□ 들여쓰기: 내어쓰기(Hanging Indent) 적용하지 않음 · 일반 공백(Space) 사용
□ 페이지 번호: - N - 형식 · 바닥글 가운데 · 12pt
□ 문서 끝: - 以 上 - 오른쪽 정렬 · 14pt
□ Word 자동 고침 옵션 해제 (파일 > 옵션 > 언어 교정 > 자동 고침 옵션)
- 곧은 따옴표를 둥근 따옴표로
- 하이픈을 대시로
- 첫 줄을 들여쓴 단락의 맨 첫 부분 간격
- 자동으로 글머리 기호 넣기
- 자동으로 번호 매기기
- 목록 항목의 첫 부분을 이전 항목과 같게 서식 적용
- 탭과 백스페이스 넣을 때 첫 부분에 왼쪽으로 들여쓰기
