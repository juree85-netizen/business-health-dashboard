5들여쓰기 & 줄바꿈 규칙
5.1  들여쓰기 설정값
레벨 기호 왼쪽 들여쓰기 내어쓰기 (Hanging) 비고
Lv 1 1. 공백 0칸 없음 직접 입력, 자동 번호 매기기 사용 안 함
Lv 2 □ 공백 1칸 없음 일반 공백(Space) 사용
Lv 3 - 공백 3칸 없음 일반 공백(Space) 사용
Lv 4 · 공백 5칸 없음 일반 공백(Space) 사용
Lv 5 ※ 공백 7칸 없음 일반 공백(Space) 사용
들여쓰기 원칙: 내어쓰기(Hanging Indent)를 적용하지 않습니다. 모든 들여쓰기는 일반 공백(Space)으로 처리합니다.
5.2  줄바꿈 정렬 시각화
│    □ 소소제목 내용이 길어서 한 줄을 넘어갈 경우에는
│       ↳ Enter로 강제 개행 후 공백 3칸(1칸+2칸 추가)으로 맞춤 정렬
│       - 세부 내용이 길어서 한 줄을 넘어갈 경우에는
│          ↳ Enter로 강제 개행 후 공백 5칸(3칸+2칸 추가)으로 맞춤 정렬
          ↑ 줄바꿈 시 해당 레벨 공백 수에 2칸을 추가하여 들여쓰기 정렬
