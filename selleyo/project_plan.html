01 기획 배경 및 핵심 해결 과제

🔍 출발점

  • 결혼·육아로 외식이 어려워진 일상
  • 가고 싶었던 맛집 음식을 집에서 재현하고 싶은 니즈
  • "집밥으로 맛집을 만들 수 없을까?" 라는 질문에서 시작

😤 기존 불편함

  • 유튜브 영상 보며 메모장에 따로 기록
  • 마트에서 재료를 기억에 의존해 구매
  • 요리 중 영상 틀고/멈추고/되감기 반복
  • 손에 재료 묻은 채로 폰 조작
🎯 핵심 해결 과제: 영상 되감기 없이, 단계별 카드만 넘기며 요리할 수 있게

04 주요 화면 상세
📊 홈 화면 — 트렌드 랭킹
  • AI가 SNS/방송/유튜브 실시간 트렌드 반영
  • 랭킹 옆 사람들이 만들어본 레시피 수 표시
  • 음식명 클릭 → 유튜브/웹 검색 링크 연결
🔗 레시피 저장 — 영상 캡처 방식 선택
  • 유튜브 링크 붙여넣기 → AI 자막 추출
  • 단계별 레시피 자동 정리
  • 핵심 장면 이미지: 저장 시 A/B 선택
옵션 방식 특징
A. 실제 프레임 영상 핵심 순간 캡처 직관적, 실제감 높음
저작권 검토 필요
B. AI 일러스트 DALL-E 단계별 재현 저작권 안전
통일감 있는 스타일
🥬 나의 냉장고 — 재료 기능 두 진입점
📦 진입점 A — 재료 먼저 (냉장고 파먹기)
  • 보유 재료 입력 → 만들 수 있는 메뉴 자동 추천
  • 재료 부족 시 "○○ 추가하면 ○○ 가능" 안내
  • 홈/탐색 화면에서 독립 진입 — 메인 기능으로 위치
🍽️ 진입점 B — 레시피 먼저 (요리 준비 흐름)
  • 레시피 선택 → 재료 확인 화면에서 보유 재료 자동 체크
  • 부족한 재료만 구매 리스트로 자동 정리
  • 요리 준비 3단계 흐름에 통합 (기존 흐름 유지)
🔗 "나의 냉장고"에 재료 등록 시 두 진입점 모두 자동 연동
🥘 요리 준비 — 3단계 구조
