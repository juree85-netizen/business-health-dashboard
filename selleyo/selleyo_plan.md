# 셀레요 (Selleyo) 앱 기획서

> **"오늘 저녁, 셀레요"**
> 셀프 레시피 요리 — 먹고 싶은 음식을 누구나 쉽게 집에서 만들 수 있게

**버전: v1.1 | 업데이트: 2026-06-12**
- 영상 캡처 방식: AI 일러스트 단독 → 실제 프레임 캡처(A) / AI 일러스트(B) 병행 옵션 추가
- 재료 기능: 두 진입점(재료 먼저 / 레시피 먼저) 병행 구조로 확장

## 1. 기획 배경
결혼/육아로 외식이 어려워진 상황에서, 가고 싶었던 맛집 음식을 집에서 재현하고 싶은 니즈에서 출발.
기존 불편함:
- 유튜브 영상을 보고 메모장에 따로 기록
- 마트에서 재료를 기억해서 구매
- 요리 중 영상을 틀고/멈추고/되감기 반복
- 손에 재료가 묻은 채로 폰을 조작
핵심 해결 과제: 영상 되감기 없이, 단계별 카드만 넘기며 요리할 수 있게

## 2. 앱 이름 및 슬로건
- 앱 이름: 셀레요 (Selleyo)
- 뜻: 셀프(Self) + 레시피(Recipe) + 요리(Cooking)
- 중의적 의미: "설레요" — 요리가 설레는 느낌
- 슬로건: "오늘 저녁, 셀레요"

## 3. 핵심 가치
- 유명 맛집 음식을 집에서 쉽게 재현
- 요리 초보자도 단계별 카드만 따라가면 완성
- 나만의 레시피를 쌓아가는 개인 요리 기록장
- 나중에는 함께 만들어가는 요리 커뮤니티

## 4. 앱 전체 흐름
1. 앱 열기 → AI 실시간 트렌드 랭킹 + 레시피 수 표시
2. 레시피 탐색 → 랭킹에서 관심 음식 발견 → 유튜브/웹 검색 링크로 확인
3. 나만의 레시피 저장 → 유튜브 링크 입력 → AI가 자막/음성 텍스트 추출 → 단계별 레시피 자동 정리 → 핵심 장면 이미지 자동 생성(단계당 1장, 옵션 A/B 선택) → 사용자 확인 및 수정
4. 나만의 레시피북 → 저장한 레시피 목록 → 공개/비공개 선택
5. 요리 준비 (3단계 구조): [요약] → [재료 체크] → [요리 시작]
6. 단계별 요리 카드 → 카드 넘기기 + 이미지(실제 프레임 또는 AI 일러스트) + 타이머

## 5. 주요 화면 상세

### 5-1. 홈 화면 — 트렌드 랭킹
- AI가 실시간으로 SNS/방송/유튜브 트렌드 반영
- 랭킹 옆에 사람들이 만들어본 레시피 수 표시

### 5-2. 레시피 저장 — 영상 캡처 방식 (A/B 선택)
- 유튜브 링크 붙여넣기
- AI 자막 추출 → 단계별 자동 정리
- 핵심 장면 이미지 1장 자동 생성: 사용자가 저장 시 두 옵션 중 선택

| 옵션 | 방식 | 특징 |
|---|---|---|
| A. 실제 프레임 캡처 | 영상의 핵심 순간을 실제 화면으로 캡처 | 직관적, 실제감 높음, 저작권 검토 필요 |
| B. AI 일러스트 | DALL-E가 해당 단계를 일러스트로 재현 | 저작권 안전, 통일감 있는 스타일 |

### 5-3. 나의 냉장고 — 재료 기능 (두 진입점)
보유 재료를 "나의 냉장고"에 등록하면 두 진입점 모두 자동 연동.

**진입점 A — "재료 먼저" (냉장고 파먹기)**
- 보유 재료 입력 → 만들 수 있는 메뉴 자동 추천
- 재료가 부족하면 "○○ 추가하면 ○○ 가능" 안내
- 독립적인 메인 기능으로 홈/탐색 화면에서 바로 접근

**진입점 B — "레시피 먼저" (요리 준비 흐름)**
- 레시피 선택 → 재료 확인 화면에서 보유 재료 자동 체크
- 부족한 재료만 구매 리스트로 자동 정리
- 요리 준비 3단계 흐름에 통합 (기존 흐름 유지)

### 5-4. 요리 준비 — 3단계 구조
① 요약 화면: 음식 대표 이미지 / 요리 시간·난이도·몇 인분 / 어울리는 상황
② 재료 확인: 재료 목록 + 체크박스 / 나의 냉장고 연동 시 보유 재료 자동 체크 / 체크 안 된 재료만 구매 리스트 자동 정리
③ 요리 시작: 단계별 카드 넘기기 / 이미지(옵션 A 또는 B) / 단계별 타이머 자동 실행

### 5-5. 나만의 레시피북
- 저장한 레시피 목록
- 나만 보기 / 공개 선택
- 공개된 레시피는 트렌드 랭킹 "레시피 수"에 반영

## 6. 검색 조건
재료 / 계절 / 시간대(낮·밤) / 상황(식사·안주·야식·간식)

## 7. 추후 확장 기능
- 재료 구매처 가격 비교 (영수증 인증 기반)
- 영수증 인증 + 포인트 시스템
- 맛집 기록 연결
- 커뮤니티 확장

## 8. 수익 모델
- 초기: 쿠팡/마켓컬리 제휴 수수료
- 중기: 프리미엄 구독 (이미지 생성 무제한, 레시피 무제한)
- 후기: 지역 시장/식자재 광고, 식품 브랜드 콜라보

## 9. 기술 스택
- 프론트엔드: React
- 백엔드/DB: Supabase
- AI 텍스트: OpenAI API
- AI 이미지 (옵션 B): DALL-E
- 영상 캡처 (옵션 A): YouTube Data API / 썸네일·프레임 추출
- 개발 도구: Cursor

## 10. 개발 단계
- 1단계(MVP): 유튜브 링크 입력 → AI 자막 추출 → 단계별 정리 → 이미지 생성(A/B 선택) → 요리 카드+타이머
- 2단계: 트렌드 랭킹 / 레시피북(공개·비공개) / 검색 필터
- 3단계: 나의 냉장고(재료 먼저 + 레시피 먼저 연동) / 재료 가격 비교 / 영수증 인증+포인트
- 4단계: 커뮤니티 / 맛집 기록 / 제휴·광고 수익화
